== Description ==

Send email with WooCommerce reports periodically.

Get your daily sales reports, monthly, yearly ...<br>
Stay informed about the finances of your online store.<br>
With just three steps:<br>
1. - Choose frequency<br>
2. - Choose report<br>
3. - Select destination emails

Official site: [eggemplo](https://eggemplo.com)

Documentation site: [documentation](https://www.eggemplo.com/document/woocommerce-email-report/)

== Installation ==

You can either install it automatically from the WordPress admin, or do it manually:

1. Unzip the archive and put the `wooemailreport` folder into your plugins folder (/wp-content/plugins/).
2. Activate the plugin from the Plugins menu.
